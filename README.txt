
# Welcome

As a Metal Gear Solid fan when I saw liquid templating language with .liquid files, I figured I gotta turn my blog around using it. This is essentially html with nothing too complicated. Supports deployment on Github Pages out of the box.

This repo is for my blog posts, its design is built from the ground up with simplicity at it's core. Inspired by the astro-ink theme (their demo link doesn't even works anymore I don't know why it feels so abandoned back in the days everyone loved it). 

I hope this repo will be useful to others. I sure hope you like the blog posts too. Changed the readme into a .txt file to match the vibe of some of the most epic repos I have seen which see themselves very lightly.
